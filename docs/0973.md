# Android 文本视图–对齐文本

> 原文：<https://www.tutorialkart.com/kotlin-android/android-textview-justify-text/>

在某些情况下，对齐 TextView 的文本有助于更好地向用户呈现数据。左侧和右侧的文本对齐使段落与其他元素和屏幕边框对齐。

**注意**:这个功能只有 Android 8.0 才有。
