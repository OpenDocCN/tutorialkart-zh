## 颤振数据表

如果表中要显示的行数较少，可以使用 DataTable。

DataTable 是一个材料设计数据表，您可以在其中显示带有列标签和行的表格。

**注意**:当且仅当你有较少的行时使用数据表，因为数据表在 UI 上布局元素需要两倍的计算。
