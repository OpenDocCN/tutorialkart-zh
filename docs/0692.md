## 在 Ubuntu 上安装 MongoDB

在 Ubuntu 上安装 MongoDB——在本教程中，我们将学习在 Ubuntu 16.04 上安装 MongoDB 社区版。对于 64 位版本的 Ubuntu 14.04 和 Ubuntu 12.04，该过程应该保持相同。

下面是我们在 Ubuntu 上安装 MongoDB**的步骤的简要概述。**

1.  导入 MongoDB GPG 公钥
2.  为 MongoDB 创建列表文件
3.  重新加载本地包数据库
4.  安装 MongoDB 包
5.  启动 MongoDB
6.  停止 MongoDB
