## C++数组

在 C++中，数组是相同类型元素的有序集合。

例如，下面是一个整数数组。

```
{7, 3, 8, 7, 2}

```

下面是一个字符串数组。

```
{"apple", "banana", "cherry"}

```

因为这些数组是有序的，所以我们可以使用 index 来访问这个数组`x`中的元素。

```
x[2] //returns element at index=2

```

其中 index 从零开始，随后的元素递增 1，直到数组结束。
