## MySQL–选择数据库

MySQL 服务器可以包含多个数据库，并且可以同时服务多个客户端。因此，当客户端连接或打开 mysql 命令提示符时，应该选择一个数据库(从现有的多个数据库中)来运行 sql 查询或操作。在本教程中，我们将学习从多个数据库中选择一个数据库。
