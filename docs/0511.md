# SAP HR–在 SAP 中定义员工组

**什么是员工小组？**

**SAP**中的员工组指定了一个雇佣类别，例如永久员工、合同工、实习生等。员工组和员工子组被认为是人员结构，是企业结构的一部分。

> SAP 中员工组的字段名是“PERSG”
