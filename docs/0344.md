# 如何在 SAP 中维护存储位置

> 原文：<https://www.tutorialkart.com/sap-mm/how-to-maintain-storage-location-in-sap/>

**SAP 中的存储位置是什么？**

存储位置是工厂的子部门，库存在[工厂](https://www.tutorialkart.com/sap-mm/how-to-define-plant-in-sap/)内进行物理存储和维护。工厂内的每个存储位置都有自己的地址。

*   要求为一个工厂保留至少一个存储位置。
*   一个工厂可以有多个存储位置。
