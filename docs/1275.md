## JavaFX 密码字段

在 JavaFX 中，PasswordField 与 TextField 相同，只是其中输入的字符被屏蔽了。

在本教程中，我们将学习如何使用 Java 在我们的 GUI 应用程序中定义一个密码字段，以及当一个事件发生时，如点击按钮，如何读取用户在密码字段中提供的密码。
