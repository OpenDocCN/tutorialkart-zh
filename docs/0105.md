## Apex 编程中的构造函数是什么？

Apex 编程中的构造函数是一个代码，是从类创建对象时调用的特殊方法。构造函数具有以下属性。

1.  方法名将与类名相同。
2.  访问说明符将是公共的。
3.  这个方法只会在建立物件时叫用一个。
4.  这用于实例化该类的数据成员。

**例如:**

```
public class TestObject {
	// the no argument Constructor
	public TestObject() {
		//code
	}
}
```
