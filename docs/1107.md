## 颤动升起按钮

在 Flutter 中，RaisedButton widget 是一个带仰角的按钮的材质设计概念。它为你的 UI 提供了沿 z 轴的维度和阴影的线索。

RaisedButton 已折旧。请参考[升高按钮](https://www.tutorialkart.com/flutter/flutter-elevatedbutton/)显示按钮。
