## Python 列表

Python 列表是元素的有序集合，可以在列表中存储任何数据类型的对象。

因为 Python List 是一个有序集合，所以可以使用 index(元素在元素中的位置)来访问元素。像 Java、C、C++等其他编程语言一样，索引从`0`开始，直到列表中的项目数。
