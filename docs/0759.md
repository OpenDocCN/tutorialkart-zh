## PostgreSQL–插入查询

要在 PostgreSQL 表中插入一行，可以按以下语法使用 INSERT INTO query。

```
INSERT INTO table_name(
	column1, column2,.., columnN)
	VALUES (value1, value2,.., valueN);
```

在哪里

*   table_name 是要在其中插入行的表的名称。
*   列 1、列 2、..，columnN 是表格的列。
*   值 1，值 2，..，valueN 是各列的相应值。
