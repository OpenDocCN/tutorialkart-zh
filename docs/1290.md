## type script if else–条件语句

TypeScript **if else** 是对 **if** 条件语句的扩展。 **else** 块是可选块，可以跟在 if 块后面。 **if else** block 根据条件将执行流程分支到两个块中的一个。

如果 **if** 关键字后的表达式返回值为真，则执行 **if** 语句后的代码块，否则执行 **else** 块后的代码块。
