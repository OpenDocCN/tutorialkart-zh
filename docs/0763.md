# 如何给 PostgreSQL 表添加列？

> 原文：<https://www.tutorialkart.com/postgresql/postgresql-add-column/>

## PostgreSQL–添加列

要向现有 PostgreSQL 表中添加新列，请使用以下 ALTER TABLE 语法。

```
ALTER TABLE table_name
	ADD COLUMN new_column_name data_type;
```
