## Ubuntu 创建用户

**创建用户**–当你在你的电脑上安装 Ubuntu 操作系统时，一个用户被创建。稍后，您可以在电脑上创建更多用户，供共享电脑的家庭成员或同事使用。

在本教程中，我们将学习如何在 Ubuntu 中创建用户。

我们使用的是在 Linux 内核版本 5.4.0-33-generic 上运行 OS Ubuntu 20.04 的 PC。
