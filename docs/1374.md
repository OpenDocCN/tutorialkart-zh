## go–字符串连接

在 Go 编程中串接字符串，我们可以使用`strings.Join()`函数。

字符串的语法。Join()函数是:

```
strings.Join(a []string, sep string)
```

其中`a`是一个字符串数组，`sep`字符串用于相邻字符串的连接之间。
