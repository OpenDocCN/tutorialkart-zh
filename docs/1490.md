## 使用 PDFBox 2.0 阅读 PDF 文档中的所有文本

在本教程中，我们将学习在 Java 程序中使用 PDFBox 2.0 库读取 pdf 文档中的所有文本。

PDF 文档可能包含文本、嵌入图像等。，作为它的内容。PDFBox 中的 PDFTextStripper 类提供了从 PDF 文档中提取所有文本的函数。
