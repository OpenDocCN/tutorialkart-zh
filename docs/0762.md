## PostgreSQL 更新

PostgreSQL 更新查询用于更新表中的列值。您可以应用 WHERE 条件，以便仅对满足 WHERE 子句中的条件的值应用 UPDATE。

更新查询的语法是:

```
UPDATE table_name
	SET column1 = value1, column2 = value2, columnN = valueN
	WHERE condition;
```

其中，列 1 的值设置为值 1，列 2 的值设置为值 2，依此类推，如 set 子句中所定义。
