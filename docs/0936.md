## 快速元组

元组用于在单个变量名下对不同数据类型的值进行分组。

下面是一个存储在名为`player`的变量中的 Swift Tuple 的例子。

```
var player = (45, "Rohit")
```

元组`player`有两个值。第一个是 Int，第二个是 String。我们可以在一个元组中混合任何数据类型。
