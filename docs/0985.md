# 使用 Kotlin 在 Android 中以编程方式创建一个新的 EditText 小部件

## 以编程方式创建 EditText

我们知道可以使用布局文件指定 EditText 小部件。但是，我们也可以以编程方式创建一个 EditText，然后在布局文件中将它作为子视图添加到特定视图中。

在本教程中，我们将学习如何在 Android 中以编程方式创建 EditText，并将该 EditText 添加到布局文件中的 LinearLayout。
