## 评论

JavaScript 注释是程序员放在脚本中的冗长文本，用于增强脚本的可读性。

解释器会忽略注释。因此，它只是为了人类阅读的目的，而不是为了执行。

根据注释的行数，JavaScript 中有两种注释。他们是

*   单行注释
*   多行注释
