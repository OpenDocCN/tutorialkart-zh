## Mongo Shell

**Mongo shell** 是 MongoDB 的一个交互式 JavaScript 接口，用于执行管理操作以及查询和更新数据。

在连接到 MongoDB 之前，请确保 MongoDB 正在运行。如果不是，[启动 MongoDB](https://www.tutorialkart.com/mongodb/install-mongodb-on-ubuntu/#start-mongodb) 。
