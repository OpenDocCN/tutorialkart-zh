## SAP ABAP 关键词

**SAP ABAP 关键字**用于识别报表的类型。每个 ABAP 语句都应该以关键字开头。ABAP 不是区分大小写的实时 JAVA 语言，预定义的单词可以是任何小写或大写。

```
WRITE 'Hello WORLD'

wRITE 'Hello World'
```
