# SAP HR–在 SAP 中定义信息组(信息组)

> 原文：<https://www.tutorialkart.com/sap-hr/define-infogroups-in-sap/>

在本教程中，您将逐步了解什么是 SAP 中的信息组，以及如何在 SAP HR 中定义信息组。

**什么是 SAP HR 中的信息组**

在 SAP 中执行特定操作的相关信息类型(信息类型列表)的分组称为信息组。操作将按照信息组的顺序执行信息类型。
