## JavaFX 图像视图

JavaFX ImageView 用于在 UI 中显示图像。

在本教程中，我们将学习如何使用 ImageView 类在 UI 中显示图像。

**步骤 1** :以 FileInputStream 格式读取图像，并使用该流准备一个图像对象。

**步骤 2** :将 Image 对象作为参数传递给 ImageView()构造函数。
