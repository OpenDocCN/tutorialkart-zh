## Android 文本视图–斜体文本

TextView Italic Text–要将 TextView 的文本样式设置为斜体，您可以在 XML 布局文件中将 **textStyle** 属性指定为“Italic ”,或者使用 **setTypeface()** 方法在 Kotlin 文件中动态更改文本样式。

在本教程中，我们将学习布局文件方法和 Kotlin 线方法来改变文本视图的文本样式为斜体。
