## Node.js MySQL 结果对象

当在 Node.js 中执行 MySQL 查询时，一个名为 Result Object 的对象被返回给回调函数。

Result 对象包含结果集或属性，这些结果集或属性提供有关 MySQL 服务器中查询执行的信息。
