# Scala–从控制台读取数字作为输入–整数，浮点

## 从控制台输入读取数字 Scala

如果你正在用 Scala 开发一个控制台应用程序，你可能需要从用户那里读取输入。

在本教程中，我们将学习如何从控制台读取用户输入的数字。

为了从控制台输入中读取一个数字，我们使用了方法`scala.io.StdIn.readInt()`，`scala.io.StdIn.readFloat()`。
