## 快速集合

Swift Set 是唯一元素的无序集合。

存储在集合中的元素没有顺序。因此，我们不能使用索引来访问集合中的元素。

下面是一个名为`fruits`的集合的简单例子。

```
let fruits: Set = ["apple", "banana", "cherry", "mango"]
```
