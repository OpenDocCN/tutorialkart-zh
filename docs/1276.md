## JavaFX DatePicker

GUI 中的 JavaFX DataPicker 允许用户输入日期值或单击日历图标并从弹出菜单中选择日期。

在本教程中，我们将学习如何初始化一个 DatePicker 控件并在 GUI 中显示它，然后添加一个动作监听器来获取用户使用 DatePicker 选择的值。
