## C++字符串

在 C++中，String 是一个类类型`std::string`的对象。这种类型的对象代表一个序列或字符串，因此称为字符串。

下面是一个字符串的示例。

```
"Hello World"

```

该字符串包含 11 个字符。在 C++中，字符串常量用双引号括起来。

`string`关键字用于指定变量的数据类型。

```
string s = "Hello World";

```

在本教程中，我们将通过例子介绍最常用的 C++字符串操作。
