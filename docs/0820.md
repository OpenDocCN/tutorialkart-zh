# Java 字符串

> 原文:[https://www.tutorialkart.com/java/java-string/](https://www.tutorialkart.com/java/java-string/)

字符串是一系列字符。例如，`"Hello World"`是一个字符串。

Java 通过 String 类支持字符串。它提供了创建字符串的构造函数和对字符串执行函数的方法。
