## Java 数组

在 Java 中，数组是项目的集合，其中的项目是有序的。数组是不可变的，这意味着我们不能在数组中添加或删除元素。

例如，赋给变量`arr`的整数数组是

```
int arr[] = {2, 4, 6};
```

字符串数组是

```
String arr[] = {"apple", "banana", "cherry"};
```
