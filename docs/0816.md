## Java 决策

决策语句是那些让我们根据条件执行代码块的语句。这有助于我们在 Java 应用程序中实现决策逻辑。

Java 支持以下条件语句。

*   如果语句
*   If-Else 语句
*   If-Else-If 语句
*   交换语句
