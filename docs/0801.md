## Linux 创建文件

要在 Linux 文件系统中创建一个文件，可以在终端中使用 mkdir 命令，或者在文件名后面加上`>`。在 Linux 中，有许多方法可以通过终端创建文件。

在本教程中，我们将学习如何在两个过程中创建一个文件。在第一个过程中，我们将使用触摸命令。在第二个过程中，我们将使用`>`。
