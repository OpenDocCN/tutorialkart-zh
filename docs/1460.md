## 卡夫卡——创造主题

所有关于 Kafka 主题的信息都存储在 Zookeeper(集群管理器)中。对于每个主题，您可以指定复制因子和分区数量。主题由其名称来标识。在创建新的 kafka 主题时，所有这些信息都必须作为 shell 脚本/kafka-topics.sh 的参数提供。
