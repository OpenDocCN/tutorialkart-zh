## 戈里普斯

goroutine 是由 Go 运行时管理的轻量级线程。

要启动一个 goroutine，在进行函数调用之前使用关键字`go`，如下所示。

```
go somefunc(arg1, arg2)
```

当执行上述语句时，语句本身在当前的 goroutine 中执行，但是`somefunc(arg1, arg2)`在新的 goroutine 中执行。
