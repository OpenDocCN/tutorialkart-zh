### 关于

Java 是一种广泛用于开发软件的编程语言。本 Java 教程旨在帮助初学者开始学习 Java 编程语言。

Java 编程语言是开发人员中最流行的语言之一。大多数生产服务器、普通台式机、工作站、智能手机等。，都是 Java 驱动的。
