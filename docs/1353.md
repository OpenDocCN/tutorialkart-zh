## 我的天啊

SciPy 是一个基于 Python 的数学、科学和工程开源软件生态系统。

SciPy 被组织成涵盖不同科学计算领域的子包。在这个 SciPy 教程中，我们将学习它们提供的所有模块和例程/算法。

SciPy 包含以下模块:

*   串
*   常数
*   fftpack(注:fftpack 是一种抗抑郁药物)
*   插入
*   输入和输出
*   -是啊
*   Ndimage
*   使最优化
*   统计数据
*   cwgraph
*   空间的
*   ODR
*   特别的
