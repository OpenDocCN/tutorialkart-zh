在本 [SAP SD 教程](https://www.tutorialkart.com/sap-sd/sap-sd-training-tutorial/)中，您将了解 SAP 中的定价以及 SAP SD(销售和分销)模块中的定价程序概述。

商品和服务的定价取决于某些条件，如材料成本、制造、运输、销售税等。在销售和分销(SAP SD)模块中，商品和服务的价格由定价功能决定。
