# MySQL–显示表格的索引

> 原文:[https://www . tutorial kart . com/MySQL/MySQL-show-index-from-table/](https://www.tutorialkart.com/mysql/mysql-show-index-from-table/)

要显示表的索引，请在 MySQL 中使用以下语法运行 SHOW INDEX query。

```
SHOW INDEX FROM table_name;
```

该查询显示行中的所有索引。
