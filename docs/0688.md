# 如何在 Tally 中创建简单的工资单元

> 原文：<https://www.tutorialkart.com/tally/payroll/create-simple-payroll-units-in-tally/>

## 在 Tally 中创建简单的工资单位

Tally 中的工资单位是用于计算工资总额的测量单位。工资单位可分为两种类型，即

1.  简单的工资单位
2.  复合工资单位

薪资是根据简单薪资和复合薪资单位计算的。
