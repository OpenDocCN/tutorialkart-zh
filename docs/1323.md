# 朱莉娅算术运算符

> 原文:[https://www . tutorial kart . com/Julia/Julia-算术-运算符/](https://www.tutorialkart.com/julia/julia-arithmetic-operators/)

在这个 Julia 教程中，我们将学习 Julia 中有哪些算术运算符，以及如何通过例子来使用它们。

所有基本数字类型都支持以下算术运算符:

<figure class="wp-block-table">

| 表示 | 名字 | 描述 |
| `+x` | 一元加号 | 身份操作 |
| `-x` | 一元减操作 | 将值映射到它们的加法逆 |
| `x + y` | 二进制加号 | 执行加法 |
| `x - y` | 二进制减 | 执行减法 |
| `x * y` | 倍 | 执行乘法 |
| `x / y` | 划分 | 执行除法 |
| `x ÷ y` | 整数除法 | x / y，截断为整数 |
| `x \ y` | 反向除法 | 相当于`y / x` |
| `x ^ y` | 力量 | 将`x`提升到`y`次方 |
| `x % y` | 剩余物 | 相当于`rem(x,y)` |

</figure>
