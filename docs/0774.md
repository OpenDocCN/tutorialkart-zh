## MySQL–显示数据库

在 MySQL 中可以同时创建和使用许多数据库。通常一个数据库服务于一个应用程序。不同的应用程序可以连接到同一个 MySQL 服务器来访问不同的数据库。在 MySQL 服务器中可以存在多个数据库的背景下，我们将学习如何在 MySQL 中查看或显示现有的数据库。
