## JavaFX 复选框

GUI 中的 JavaFX 复选框允许用户切换设置、属性或某些值。

在本教程中，我们将学习如何初始化一个复选框控件并在 GUI 中显示它，然后添加一个动作监听器来知道复选框是否被选中。
