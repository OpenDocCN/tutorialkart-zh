## MySQL–向索引添加列

索引用于以更高的性能访问表中的行。

假设您有一个很大的表，并且有一个查询，在该查询中，您根据一个列提取行。如果该列不是主键，但是您经常使用它来过滤行，那么最好将该列添加到索引中。

要向表的索引中添加一列，下面是 SQL 查询的语法。

```
ALTER TABLE table_name ADD INDEX index_name (column_name);
```
