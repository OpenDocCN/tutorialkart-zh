## MySQL–重命名列

通常情况下，您创建一个带有一些列名的表，然后由于项目中的一些更改请求或其他原因，您可能必须更改列名。在这个 MySQL 教程中，我们将学习重命名 MySQL 表中的一列。

要重命名 MySQL 中的列，请使用以下语法:

```
ALTER TABLE table_name CHANGE old_column_name new_column_name datatype(length);
```
