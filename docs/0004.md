## 什么是 Salesforce？

Salesforce 是基于云的技术，是全球最大的基于 web 的软件和云计算公司之一，由前甲骨文高管 Marc Benioff、Parker Harris、Dave Moellenhoff 和 Frank Dominguez 于 1999 年创立，被称为 ***【客户关系管理(CRM)】***产品。
