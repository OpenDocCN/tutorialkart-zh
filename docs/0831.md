## C++ If Else 语句

在 C++中，if else 语句用于执行语句的条件执行。基于条件的结果，决定执行一个块。

在本教程中，我们将学习不同形式的 **if else** 语句，它们的语法以及详细的解释和例子。

在 C++中，If Else 语句可以以三种形式出现。它们是:

*   如果语句
*   If-Else 语句
*   If-Else-If 语句
