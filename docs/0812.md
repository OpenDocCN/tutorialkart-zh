## Java–数据类型

Java 有两种数据类型。它们是原始数据类型和用户定义的数据类型。

在本教程中，我们将借助例子详细了解基本数据类型。用户定义的数据类型是类，我们将在 [Java 面向对象编程](https://www.tutorialkart.com/java/java-oop/)概念教程中讨论这些。
