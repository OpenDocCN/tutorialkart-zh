## Android Jetpack 合成–映像

图像合成功能用于在 Android 应用程序中显示图像。

从可绘制资源中组合显示图像的最基本的图像代码如下所示。

```
Image(painter = painterResource(id = R.drawable.flower), contentDescription = null)
```

我们还可以使用可选参数修改形状或其外观。
