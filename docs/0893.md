# r 工作目录

> 原文：<https://www.tutorialkart.com/r-tutorial/r-working-directory/>

## 获取/设置 R 中的工作目录

在本教程中，我们将学习如何使用示例 R 脚本设置 R 工作目录并获取 R 工作目录的值。

R 工作目录是 R 工作区的目录。R 工作空间中的任何文件都可以在 R 命令中引用，而无需指定任何相对路径。在处理外部输入文件或输出文件时，了解 R 工作空间有助于减轻工作量。
