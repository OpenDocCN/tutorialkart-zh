# fast text Python–学习单词表示

我们将学习如何通过使用无监督学习技术训练单词向量，让模型使用 Python 中的 FastText 学习单词表示。

## 学习快速文本中的单词表示

对于使用机器学习的训练，单词和句子可以用一种更数字和更有效的方式来表示，称为单词向量。FastText 提供了学习这些单词表示的工具，可以提高文本分类等的准确性。
