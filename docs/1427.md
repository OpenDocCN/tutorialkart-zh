## 将输入文本文件读给 RDD

要将一个输入文本文件读入 RDD，我们可以使用 SparkContext.textFile()方法。

在本教程中，我们将学习 SparkContext.textFile()方法的语法，以及如何在 Java 和 Python 示例的帮助下，在 Spark 应用程序中使用将数据从文本文件加载到 RDD。
