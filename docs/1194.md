Node.js 解析 JSON——解析 Node.js 中的 JSON 数据，可以使用 JavaScript 引擎的 **JSON.parse()** 函数。

在本教程中，我们将通过示例程序学习如何使用 JSON.parse()函数解析给定的 JSON 字符串。
