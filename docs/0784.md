# 增加 MySQL 中列的大小

## MySQL–增加列大小

您是否创建了一个具有特定大小的列，是否现在增加？在本 [MySQL 教程](https://www.tutorialkart.com/mysql/)中，我们将学习如何修改 MySQL 表中一列的大小。

要更改列大小，请使用 ALTER TABLE query，如下所示:

```
ALTER TABLE table_name MODIFY column_name datatype
```
