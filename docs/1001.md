## Kotlin Android 警报对话框

Android AlertDialog 类用于显示一个对话框，用正反按钮提醒用户。正按钮用于继续指定的动作。否定按钮用于消除警告动作。您可以提供自己的自定义代码时，积极或消极的按钮被点击。

AlertDialog 出现在活动布局的顶部。除了“报警”对话框之外，您不能实际访问“活动”的任何其他 UI 组件。
