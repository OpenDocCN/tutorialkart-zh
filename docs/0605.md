# SAP MM–为功能位置创建结构指示器

> 原文：<https://www.tutorialkart.com/sap-pm/create-structure-indicator-functional-locations/>

## 如何在 SAP 中为职能位置创建结构指示器

在本活动中，您将在 SAP PM(工厂维护)模块中为功能位置和参考位置创建结构指示符。结构指示符将 SAP 中功能位置的结构指定为

*   数字层级
*   密钥长度级别
*   分隔符(-)。
