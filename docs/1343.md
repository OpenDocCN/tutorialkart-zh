## Python 运算符

Python 中有七种运算符。他们是

*   算术运算符
*   按位运算符
*   赋值运算符
*   比较运算符/关系运算符
*   标识运算符
*   成员运算符

在本教程中，我们将通过例子来研究这些操作符。
