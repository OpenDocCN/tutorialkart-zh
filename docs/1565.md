## CSS 边框半径属性

CSS border-radius 属性设置元素的圆角半径。

可以用不同的方式指定四个角的边框半径。我们将通过例子逐一介绍它们。

[![](img/925da31b32d6bc3827932f6c8afb11bb.png)](https://www.tutorialkart.com/)

如果指定了 border，则边界也会应用半径。
