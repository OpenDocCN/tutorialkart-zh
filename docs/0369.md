## 如何在文档级别为 RFQ 定义屏幕布局

在本活动中，您将在 SAP 中为 RFQ(询价)定义文档级别的屏幕布局。对于采购单据和采购申请，您需要使用字段选择键更新字段的输入属性。

文档配置方式的屏幕布局定义与 PR 的[文档级屏幕布局相同。](https://www.tutorialkart.com/sap-mm/define-screen-layout-at-document-level/)
